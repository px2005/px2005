Foodgram project

A single page application based on the React framework that will interact with the API (Django REST Framework) through a convenient user interface. This is a non-commercial online service. On this service, users can publish recipes, subscribe to publications of other users, add their favorite recipes to the "Favorites" list, and before going to the store, download a summary list of products needed to prepare one or more selected dishes

Stack: Python, Django, Django REST Framework, PostgreSQL, Nginx, gunicorn, docker, Yandex.Cloud.
